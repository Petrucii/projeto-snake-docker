# Snake Game com Backend em Docker 🐍🐳

Este projeto demonstra uma arquitetura de microserviços simples: um jogo cliente (Pygame) integrado a uma API de ranking (Flask) rodando em containers.

## 🛠️ Tecnologias
- **Frontend**: Python & Pygame
- **Backend**: Python & Flask
- **Infraestrutura**: Docker & Docker Compose

## 🚀 Como Executar

### 1. Subir o Servidor de Ranking (Docker)
Abra o terminal na raiz do projeto e execute:
```bash
docker-compose up -d
